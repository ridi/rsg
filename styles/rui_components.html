Button

default 값에 대응하는 modifier 는 생략이 가능합니다. 따라서 .RUIButton.RUIButton-color-gray.RUIButton-fill-true.RUIButton-size-medium {...} 와 .RUIButton {...} 는 동일합니다.

  • RUIButton
  • RUIButton-color-{ gray(default) / blue }
  • RUIButton-fill-{ true(default) / false }
  • RUIButton-size-{ small / medium(default) / large }
